# Task Manager

A clean, responsive, and web-based task management tool designed to streamline daily productivity and item tracking[cite: 1]. 

## 🚀 Features

- **Persistent Storage:** Utilizes browser **Local Storage APIs** to ensure your tasks remain securely saved across sessions without requiring a backend database[cite: 1].
- **Responsive Design:** Structured with clean, semantic front-end components to deliver an intuitive and user-friendly interface on both desktop and mobile devices[cite: 1].
- **Real-Time Tracking:** Easily add, manage, and keep track of your daily goals in real time[cite: 1].

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript[cite: 1]
- **Storage:** Web Storage API (LocalStorage)[cite: 1]
